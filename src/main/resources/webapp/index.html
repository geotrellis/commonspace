What is GeoTrellis Transit?

GeoTrellis Transit is a set of web services and graph building tools that uses the new Network functionality in GeoTrellis to answer travel time questions. The transit graphs are built using GTFS and Open Street Map data, and includes functionality for querying the graph for walking, biking, and public transit travel.
